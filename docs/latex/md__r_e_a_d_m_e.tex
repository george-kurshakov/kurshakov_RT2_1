\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysubsubsection{state\+\_\+machine}\label{md__r_e_a_d_m_e_autotoc_md4}
A node managing the robot behaviour. It receives commands from {\bfseries{user\+\_\+interface}} via a service and implements a simple finite state machine. It can request the {\bfseries{random\+\_\+position\+\_\+server}} for a random position, send this position as a goal for {\bfseries{\mbox{\hyperlink{namespacego__to__point}{go\+\_\+to\+\_\+point}}}} action and cancel the goal if requested.\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysubsubsection{random\+\_\+position\+\_\+server}\label{md__r_e_a_d_m_e_autotoc_md5}
A node implementing a service for generating random positions. Provides the {\bfseries{state\+\_\+machine}} with random positions for {\bfseries{\mbox{\hyperlink{namespacego__to__point}{go\+\_\+to\+\_\+point}}}}.\hypertarget{md__r_e_a_d_m_e_autotoc_md6}{}\doxysubsubsection{go\+\_\+to\+\_\+point}\label{md__r_e_a_d_m_e_autotoc_md6}
A node implementing an action for reaching the goal position using a publisher on topic $\ast$/cmd\+\_\+vel$\ast$ and a subscriber on topic $\ast$/odom$\ast$. {\bfseries{state\+\_\+machine}} is able to send a goal and cancel it. Once the goal is cancelled, the robot stops immediately.\hypertarget{md__r_e_a_d_m_e_autotoc_md7}{}\doxysubsection{Instructions to run the code}\label{md__r_e_a_d_m_e_autotoc_md7}
All the instructions can be found in the {\bfseries{interface.\+ipynb}} Jupyter Notebook file.
